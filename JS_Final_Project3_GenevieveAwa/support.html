How to Play the Game

Objective

The goal of the game is to correctly guess the superhero based on the incomplete puzzle shown on the screen.

How to Play

  1. Look at the Puzzle:
    • At the start of each level, you'll see a puzzle with some missing pieces.
    • Use the visible parts of the image as clues to identify the superhero.
  2. Make Your Guess:
    • Type the name of the superhero in the input box provided.
    • Click the Submit button to check your guess.
  3. Hints:
    • You have 3 attempts to guess the superhero's name correctly.
    • For every incorrect guess, a random missing piece of the puzzle will be revealed to give you more clues.
  4. Winning the Level:
    • From there, you can move to the next level or quit the game.
    • If you guess the superhero's name correctly, you’ll be taken to the completed puzzle page, where:
      • The full puzzle will be revealed.
      • You’ll learn more about the superhero’s story.
  5. Game Over: If you use all 3 attempts without guessing correctly, the game will reveal the completed puzzle and the superhero's name. You can still proceed to the next level or retry the current one.
    • If you use all 3 attempts without guessing correctly, the game will reveal the completed puzzle and the superhero's name.
    • You can still proceed to the next level or retry the current one.

Tips for Success

  • Pay close attention to the visible puzzle pieces—they often contain distinctive features of the superhero!
  • If you’re stuck, don’t worry—each failed attempt reveals more of the puzzle to help you out.

Navigation

  • Use the Quit button to return to the home page.
  • Use the Next button on the completed page to progress to the next level.

Need More Help?
